# Final-Fight

* 该游戏模拟的红白机经典游戏快打旋风。
* 双人版快打旋风，
* 同时砍杀两个boss，
* 杀到手软的小兵，
* 让你过足手瘾！

* 编译环境：vs2019 + EasyX_v20210730。

* 项目地址链接：https://codebus.cn/xiongfj/fc-final-fight
