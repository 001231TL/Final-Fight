# Final-Fight

该游戏模拟的红白机经典游戏快打旋风。
双人版快打旋风，
同时砍杀两个boss，
杀到手软的小兵，
让你过足手瘾！

编译环境：vc6.0 / vs2010 + EasyX_v20130322(beta)。

项目地址链接：https://codebus.cn/xiongfj/fc-final-fight
